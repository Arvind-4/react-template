## ğŸ“¦ Getting Started:

1. Clone the project and make it your own.

```bash
mkdir ~/Dev
cd ~/Dev
git clone https://github.com/Arvind-4/react-template.git
cd react-template
```

2. Install packages

```bash
pnpm install
```

3. Run dev server

```bash
pnpm dev
```

4. Build and Start the Docker File:

```bash
docker compose up --build -d
```
